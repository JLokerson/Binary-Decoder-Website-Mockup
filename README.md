# Binary Decoder Website

This website takes a photo, image file, or live video feed and translates any binary text into ASCII.

## Dependencies for Running

To start, the following needs to be installed (CAN CHANGE):

- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- [npm](https://www.npmjs.com/) (comes with Node.js)
- Any additional dependencies listed in `package.json`

## Notes

- If you encounter issues, ensure Node.js and npm are installed and up to date.
- Check `package.json` for additional scripts or configuration.
- For live video feed functionality, make sure your browser allows camera access.
